### Geographic Analysis of Out-of-Hospital Cardiac Arrest Incidents

Comprehensive GIS analysis created for Avive Solutions to examine out-of-hospital cardiac arrest (OHCA) incidents throughout Dothan, Alabama.

#### Highlights

### S&P 500 Sector Performance Analysis

End-to-end analytics pipeline analyzing five years of S&P 500 market data across all eleven sectors.

#### Highlights

- Built a Python ETL pipeline to collect and process 615,000+ records
- Loaded and queried data using SQLite
- Developed advanced SQL queries using CTEs and window functions
- Created a four-page interactive Power BI dashboard

#### Key Finding

Every market sector experienced negative average daily returns during the Federal Reserve's rate-hiking transition, while Information Technology and Consumer Discretionary sectors showed the strongest recovery trends.

**Tools:** Python • SQL • SQLite • Power BI • yfinance
